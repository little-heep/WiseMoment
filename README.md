# MomentWise 用户手册

MomentWise 是一款桌面专注与任务管理工具，核心包含番茄钟、任务清单、数据统计、托盘控制和中英文界面切换。

## 1. 主要功能

- 番茄专注：支持自定义时长（1-180 分钟）、开始/暂停/继续/停止。
- 白噪音：可开启白噪音、添加本地音频、删除音频、在专注时自动播放。
- 小球模式：专注运行时可自动收缩为小球，减少桌面干扰。
- 任务清单：添加任务、设置截止时间、完成勾选、删除任务。
- 数据统计：查看区间专注时长、完成任务数、有效天数、日均数据。
- 系统托盘：支持托盘菜单快速显示/隐藏、控制番茄钟、退出程序。
- 语言切换：支持中文/英文，切换后自动重启并应用语言。

## 2. 快速上手（普通用户）

1. 启动程序后，默认进入「专注」页面。
2. 设置专注时长，点击「开始专注」。
3. 如需环境音，勾选「白噪音」，并在音频下拉中选择或添加音频文件。
4. 进入「任务清单」页面可创建待办事项并设置截止时间。
5. 进入「专注统计」页面可按日期范围查看专注与任务完成情况。

## 3. 界面与操作说明

### 3.1 顶部按钮

- 置顶按钮：切换窗口是否始终置顶。
- 小窗按钮：隐藏主窗口并切换到小球模式。
- 设置按钮：打开设置菜单（透明度、语言、快捷键信息）。
- 最小化按钮：最小化窗口。
- 关闭按钮：关闭到系统托盘（不是完全退出）。

### 3.2 设置菜单

- 窗口透明度：可调节界面透明度。
- 语言：`中文` / `English`。
- 语言切换行为：选择后程序会自动重启，无需手动重启。

### 3.3 番茄钟页面

- 时长范围：`1-180` 分钟。
- 支持状态：开始、暂停、继续、停止。
- 专注结束后，会通过系统托盘弹出完成提醒。

### 3.4 任务清单页面

- 新增任务：输入标题并设置截止时间。
- 完成任务：勾选后将计入统计数据。
- 删除任务：可直接移除任务。
- 任务排序：未完成优先、超期优先、截止时间更早优先。

### 3.5 统计页面

- 支持自定义开始/结束日期。
- 展示指标包括：
- 专注总时长
- 完成任务数
- 有效天数
- 日均任务
- 日均专注与任务洞察

## 4. 快捷键（Windows）

- `Ctrl+Alt+M`：显示/隐藏主窗口
- `Ctrl+Alt+P`：开始/暂停/继续番茄钟

## 5. 托盘使用说明

- 单击托盘图标：显示/隐藏主窗口。
- 托盘菜单支持：
- Show（显示）
- Hide（隐藏）
- Start / Pause / Resume / Stop（番茄钟控制）
- Quit（退出程序）

## 6. 数据存储与配置

程序会自动保存配置和数据，主要包括：

- `settings.json`：界面语言、透明度、置顶、番茄时长、白噪音设置
- `momentwise.sqlite`：任务与统计主数据库
- `audio/`：白噪音音频目录

Windows 下通常位于应用数据目录（`AppDataLocation`，一般在 `%APPDATA%` 下的 `MomentWise` 相关目录）。

## 7. 构建与开发（可选）

如果你是开发者，可在 `D:\QtProject\MomentWise` 下执行：

```bash
cmake --preset qt515-mingw32
cmake --build --preset qt515-mingw32 -j
```

如需重新生成翻译文件（`.qm`）：

```bash
"D:/download/Qt/5.15.2/mingw81_32/bin/lrelease.exe" i18n/MomentWise_zh_CN.ts -qm i18n/MomentWise_zh_CN.qm
"D:/download/Qt/5.15.2/mingw81_32/bin/lrelease.exe" i18n/MomentWise_en_US.ts -qm i18n/MomentWise_en_US.qm
```

## 8. 常见问题

1. 点击关闭后窗口不见了，是不是退出了？
不是，程序会继续在系统托盘运行。请点击托盘图标恢复窗口，或在托盘菜单中选择 Quit 退出。

2. 切换语言后为什么会重新打开程序？
这是正常行为。MomentWise 会自动重启以确保所有界面文本完整切换。

3. 快捷键没有反应怎么办？
请确认当前系统没有其他程序占用相同组合键（`Ctrl+Alt+M` / `Ctrl+Alt+P`）。
